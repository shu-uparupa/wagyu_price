# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

- リモートリポジトリはサーバにありチーム全員が共有できるリポジトリ。
- ローカルリポジトリは個人のPCにあり、個人で操作できるリポジトリ。


## プッシュとマージの違いは何でしょうか？

- プッシュはローカルからリモートのリポジトリにデータを押し出すもの
- マージはブランチからブランチにデータを反映させるもの。

## コミットとプッシュの違い

- コミットは変更した内容をローカルリポジトリに反映
- プッシュは変更内容をローカルからリモートのリポジトリに反映


## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

- 変更を行った場所、変更した理由がぱっと見でわかりやすいように記載する。


## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

- プッシュする前にリーダーなどの第三者が変更内容を確認・修正できるかできないか


## コンフリクトを起こしてしまった場合、どう対処すべきですか？

- 先に変更したコードを反映する
- 後に変更したコードを反映する
- その両方を反映させてから修正する。
